# 每个角色代表一方势力组织
<br>
焦大： 东厂（伴朱棣出生入死，监察百官，被埋没导致明朝灭亡）
刘姥姥： 李自成（汉高祖农民军，因蝗灾起义，睡过宝玉的床）
通灵宝玉： 朝堂实权（传国玉玺，王莽篡位金镶玉，畸笏补天石）
贾宝玉/贾璋： 明朝子民（天下人心，心系朱元璋，乞丐和尚）
甄宝玉： 朱由崧（弘光帝，明安宗，顽劣厌学，“女儿是木做的”）
林黛玉： 朱由检（崇祯帝，明思宗，幼年丧母，当17年国君）
妙玉： 礼部（庙宇，太庙祭祀，佛教网络，神宫监宦官集团）
茗玉： 多尔衮（建州卫正白旗，谎称吊丧，红衣女鬼雪抱柴）
薛宝钗： 后金（金又，厚颜无耻，赖在东北，勾结东林党）
薛宝琴： 朝鲜（羁縻女真各部，上王和太上王，祭祀明朝）
甄士隐/甄费： 钱谦益（忍辱负重，东林党复社成员，迁都南京）
贾雨村/贾化： 江南商贾（东林党，朝堂士大夫集团，勾结满洲人）
林如海： 盐税来源（东林党早期成员，清流团体，巡盐御史）
赵姨娘： 张献忠（乡绅豪强，反叛当土匪，大西军苛政征税）
秦可卿： 朱常洛（泰昌帝，明光宗，纵欲过度，棺木国葬规格）
史太君： 朱翊钧（万历帝，明神宗，怠政为明朝覆亡埋下伏笔）
贾敬： 嘉靖帝（朱厚熜，明世宗，炼丹修道）
王夫人： 朱由校（天启帝，明熹宗，木匠）
王熙凤： 魏忠贤（九千岁，文盲，秉笔太监，和宝玉独处过）
贾元春： 袁崇焕（元宵爆竹，虎年炮仗，兵部尚书，太保从一品）
贾迎春： 后宫（包括太子和皇子，后金宫廷风月质子，产后赴死）
贾探春： 郑森（朱成功，三老爷，踏青节收台湾，施琅自杀自灭）
贾惜春： 画家（石涛朱若极，牛石慧朱道明，八大山人朱耷）
袭人/花珍珠： 朱彝尊（文官，择膏粱，枕席之欢后被罢官）
晴雯： 熊廷弼（武将，字飞白，云；孙承宗；孙传庭）
贾瑞/贾天祥/薛广： 洪承畴（假文天祥，假传殉国实则降清）
宝蟾： 康熙（康麻子，蟾宫折桂）
夏金桂： 吴三桂（嫦娥花，字月所，自许宋太祖灭南唐称帝）
甄英莲/香菱/秋菱： 永历皇帝（朱由榔, 被吴三桂杀害成了丘陵）
冯渊： 莽达（缅甸想与南明结盟，莽白被孝庄收买弑兄夺位）
薛蟠： 孝庄（大玉儿，文蝗后，削藩的最早及最有力支持者）
李纨： 郑芝龙（南明太师，无才便有德，他乡做故乡，回光返照）
史湘云： 前明文官（王士元养父，严绳孙，《石头记》作者之一）
贾巧姐： 明朝血亲（名曹雪芹，四皇子朱慈炤，被李自成救走）
